About the Project

HAGORAGI is a custom AI assistant based on OpenClaw.

What is OpenClaw?

An open-source framework for creating AI assistants. It allows integration with OpenAI, Anthropic, local models, and extending capabilities with skills.

What is HAGORAGI?

HAGORAGI is our customized instance of OpenClaw deployed on KVM8 VPS. It includes integrations with Telegram, Notion, GitHub Pages, and automatic synchronization.

Architecture

Docker with OpenClaw gateway. Cron scripts for synchronization. Telegram Bot as mobile interface. Everything versioned on GitHub.
