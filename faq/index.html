Preguntas Frecuentes

Respuestas a las preguntas más comunes sobre OpenClaw y el workspace.

¿Qué es OpenClaw?

OpenClaw es un framework open-source para crear asistentes de IA personalizados. Permite integrar diferentes proveedores de LLM (OpenAI, Anthropic, modelos locales) y extender funcionalidades mediante skills.

¿Cómo se sincroniza con GitHub?

El sistema utiliza scripts automatizados que se ejecutan cada 5 minutos via cron. El script sync-from-github.sh trae cambios del repositorio y backup-to-github.sh sube los cambios locales.

¿Por qué no recibo notificaciones de Telegram?

Verifica que el archivo scripts/notify.env existe con las variables TELEGRAM_NOTIFY_CHAT_ID y TELEGRAM_NOTIFY_BOT_TOKEN configuradas correctamente. Consulta el RUNBOOK.md para más detalles.

¿Cómo añado un nuevo skill?

Crea una carpeta en skills/ con un archivo SKILL.md que describe las instrucciones para el asistente. Opcionalmente puedes incluir scripts en scripts/ que se ejecuten con la herramienta exec.

¿Cómo actualizo desde upstream?

Ejecuta scripts/update-from-upstream.sh para traer los últimos cambios del repositorio oficial de OpenClaw. Este script hace merge con tu workspace personalizado.
