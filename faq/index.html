Preguntas Frecuentes

Respuestas a las preguntas más comunes.

¿Qué es HAGORAGI?

HAGORAGI es un asistente IA personalizado basado en OpenClaw. Incluye integraciones con Telegram, Notion, y un sistema de sincronización con GitHub.

¿Cuál es la diferencia entre HAGORAGI y OpenClaw?

OpenClaw es el framework base (upstream). HAGORAGI es nuestra versión personalizada con skills adicionales, integraciones específicas y configuración para VPS KVM8.

¿Cómo se sincroniza con GitHub?

Scripts automatizados via cron cada 5 minutos. sync-from-github.sh trae cambios y backup-to-github.sh sube cambios locales.

¿Por qué no recibo notificaciones de Telegram?

Verifica que scripts/notify.env tiene TELEGRAM_NOTIFY_CHAT_ID y TELEGRAM_NOTIFY_BOT_TOKEN. Consulta el RUNBOOK.

¿Cómo añado un nuevo skill?

Crea una carpeta en skills/ con un SKILL.md. Opcionalmente incluye scripts que se ejecuten con exec.
