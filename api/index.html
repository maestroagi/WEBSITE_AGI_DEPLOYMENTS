API Reference

Documentación técnica de la API de OpenClaw y endpoints disponibles.

Endpoints Principales

POST /api/chat

Envía un mensaje al asistente y recibe una respuesta.

{
  "message": "Tu mensaje aquí",
  "session_id": "opcional-session-id",
  "stream": false
}

GET /api/sessions

Lista las sesiones activas del usuario.

POST /api/skills/{skill}/execute

Ejecuta una acción específica de un skill.

Autenticación

La API utiliza tokens Bearer para autenticación. Incluye el header Authorization: Bearer YOUR_TOKEN en todas las peticiones.
